SYSTEM REQUIREMENTS
----------------
1. Data allocation in a Harddisk > 1GB

2. Computer memory > 1GB

3. Processor Pentium 4 or latest version

4. Windows XP Operating System, Windows Vista, Windows 7 or latest version,
   Linux variant Debian (UBuntu, Mint) and Linux variant RedHat (RHEL, CentOS, ClearOS)


INTRO
-----------
1. JIBAS School Information System could be access with a web browser at http://localhost/jibas/.
   It is recommended using Mozilla Firefox >= 3.6.

2. First time login, use superuser login: "jibas" as username and "password" as password.

3. Reconfig logo and school identity at c:\YIM\JIBAS\xampp\htdocs\jibas\include\school.config.php

4. Reconfig path for the application at c:\YIM\JIBAS\xampp\htdocs\jibas\include\application.config.php
